# Laravel Job Portal

## Install

- Cloning
```
git clone git@gitlab.com:maopedia-dev/laravel-job-portal.git
```
- cd laravel-job-portal
```
cd laravel-job-portal
```

- Composer
```
composer install
```

- Databse
```
php artisan migrate --seed
```

- Run Project
```
php artisan serve
```

-Test
```
php artisan test
```
